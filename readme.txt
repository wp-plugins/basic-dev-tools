== Installation ==

1.Download Basic Dev Tools.
2.Upload the 'basic-dev-tools' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3.Activate Basic Dev Tools from your Plugins page.
